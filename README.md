# JavaScript Calculator App

A clean and simple Calculator Web App built using HTML, CSS, and Vanilla JavaScript.
This project demonstrates DOM manipulation, event handling, and basic arithmetic logic.


## Features

* Addition, Subtraction, Multiplication, Division
* Clear (C) button
* Delete (DEL) functionality
* Percentage (%) support
* Simple and responsive UI
* Real-time calculation


## Concepts Used

* DOM Selection (querySelector, querySelectorAll)
* Event Handling (click)
* Array Conversion (Array.from)
* String Manipulation
* JavaScript eval() function
* Dynamic input handling


## How It Works

* Click on numbers → display updates
* Click operators → builds expression
* "=" → calculates result
* "C" → resets input
* "DEL" → removes last digit


## How to Run

1. Clone the repository:

   ```
   git clone https://github.com/jatin-kesharwani5/javascript-calculator-app.git
   ```

2. Open index.html in your browser
